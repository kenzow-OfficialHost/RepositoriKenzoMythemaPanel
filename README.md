# KenzoMythemaPanel
Dark fire theme for Pterodactyl Panel with installer Bash

## Requirements
- Ubuntu 16–24
- Pterodactyl Panel terinstal
- Node.js & npm

## Install Theme
bash <(curl -s https://raw.githubusercontent.com/kenzow-OfficialHost/RepositoriKenzoMythemaPanel/main/install-theme.sh)

## Update Dependencies
bash update-deps.sh

## Backup Resources
bash backup-resources.sh
